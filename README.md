# 🎬 Netflix Recommendation System

A Content-Based Movie Recommendation System built using Python, Pandas, Scikit-learn, TF-IDF, and Cosine Similarity.

## 📌 Project Overview

This project recommends movies similar to the movie selected by the user. It uses Natural Language Processing (NLP) techniques and Content-Based Filtering to generate recommendations.

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- TF-IDF Vectorization
- Cosine Similarity
- Content-Based Recommendation System
- Interactive Movie Recommendation

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab
